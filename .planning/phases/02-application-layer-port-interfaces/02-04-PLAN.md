**SoftDeleteAsync(Guid fileId, CallerContext caller, CancellationToken ct) → Task<Result<bool>>:**
1. Load file: `uow.Files.GetByIdAsync(fileId, caller.TenantId, ct)` → NotFoundError if null
2. `file.Transition(FileStatus.Deleted)` — throws InvalidStatusTransitionException if not Ready (let it bubble; Phase 6 maps to 409)
3. `uow.SaveChangesAsync(ct)`
4. `cache.RemoveAsync($"file:{fileId}", ct)` — invalidate cached metadata
5. Return Result.Success(true)

**HardDeleteAsync(Guid fileId, CallerContext caller, CancellationToken ct) → Task<Result<bool>>:**
1. Admin scope gate: `if (!caller.Scopes.Contains("storage.admin")) return Result<bool>.Failure(new AccessDeniedError("Requires storage.admin scope"))`
2. Load file: `uow.Files.GetByIdAsync(fileId, caller.TenantId, ct)` → NotFoundError if null
3. `await uow.ExecuteInTransactionAsync(async () => { await storage.DeleteAsync("files", file.StorageKey!.Value, ct); await uow.Files.HardDeleteAsync(fileId, caller.TenantId, ct); }, ct)`
4. `cache.RemoveAsync($"file:{fileId}", ct)`
5. Return Result.Success(true)

**ListFilesAsync(FileListQuery query, CancellationToken ct) → Task<Result<FileListResponse>>:**
1. `uow.Files.ListAsync(query, ct)` — returns up to query.PageSize+1 items
2. Determine if there is a next page: if count > query.PageSize, there are more results
3. Build NextCursor: use the last returned item's Id as an opaque cursor (base64-encode Guid.ToString("N") for readability). Return only PageSize items (drop the +1 sentinel).
4. Map domain Files to GetFileResponse records (DownloadUrl = null at this layer — Phase 6 wires real URL generation)
5. Return Result.Success(new FileListResponse(items, nextCursor, null))

1. Load file; NotFoundError if null or wrong tenant; check file.Status == Ready

4. `cache.SetAsync(cacheKey, new ShareRecord(fileId, caller.TenantId, expiresAt), ttl, ct)` — define ShareRecord as an internal record
